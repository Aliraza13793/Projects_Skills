## Employee Actions Guide

### 1. Starting New Employee Records
When onboarding a new employee:
- Verify designation against standard salary chart
- Collect complete employee information
- Assign appropriate salary based on designation
- Confirm account number for salary transfer
- Document all details in standard format

### 2. Stopping Employee Records
When terminating an employee:
- Mark employment status as terminated
- Specify termination date
- Process final salary calculations
- Close salary payments from next cycle
- Archive employee record appropriately

### 3. Correcting Employee Information
When correcting information:
- Identify which field needs correction
- Verify correct information from official sources
- Update only the specific incorrect field
- Maintain record of changes made
- Ensure salary alignment with designation

### 4. Editing Employee Information
When editing information:
- Determine scope of changes needed
- Verify new information is accurate
- Update all relevant fields as needed
- Check if salary adjustment is required
- Document the changes made

## Change Management Process

### For Salary Adjustments:
1. Verify if designation change warrants salary adjustment
2. Cross-check with standard salary chart
3. Calculate new salary effective from specified date
4. Update employee record with new information
5. Notify employee of changes with effective date

### For Information Corrections:
1. Obtain verified information from employee
2. Update only the incorrect fields
3. Maintain audit trail of changes
4. Confirm accuracy of updated information
5. Communicate changes to relevant departments
