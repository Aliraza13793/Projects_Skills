# Professional Announcements Skill

This skill helps create professional announcements suitable for WhatsApp groups and other communication channels, following the appropriate tone and format for educational and professional contexts.

## When to use this skill

Use this skill when creating:
- Class announcements and schedules
- Workshop or training notifications
- Meeting invitations
- Helpful resource sharing
- Project updates
- Event notifications
- Educational activity announcements

## How to use this skill

1. **Identify the announcement type** from the request (class, event, resource sharing, etc.)
2. **Gather necessary details** like timing, location, topics, links, and action items
3. **Apply appropriate greeting** based on audience (formal greetings like "Assalam u alykum" for certain audiences, or standard greetings)
4. **Follow the structure** outlined in the references for consistent formatting
5. **Include all relevant details** in a clear, organized manner
6. **Review for tone consistency** - ensure it matches the professional yet approachable style of the examples
7. **Consider mobile readability** - format for easy consumption on mobile devices (short paragraphs, bullet points, appropriate emojis)

## Announcement Structure Guidelines

Professional announcements typically follow this structure:
1. **Appropriate greeting** (adjust based on audience and context)
2. **Main topic/announcement** with clear subject
3. **Detailed information** including times, dates, locations, or other specifics
4. **Supporting information** such as links, resources, or additional context
5. **Action items** if applicable (forms to fill, materials to prepare, etc.)
